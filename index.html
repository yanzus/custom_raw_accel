Understanding Raw Accel — Natural Mode
What is Raw Accel?
Raw Accel is a Windows kernel-level driver that applies a custom acceleration curve to your raw mouse input before any game sees it. It works with any game, any DPI, and cannot be detected as cheating because it operates at the OS level.
Parameter Reference
Sens Multiplier Global speed scale. 1.0 = no change. This is applied after the accel curve. Start here to set your baseline feel.
Decay Rate How quickly the curve rises toward the limit. Higher = faster ramp. Lower = gradual, more control-friendly. Range: 0.01–0.40.
Input Offset Dead zone before acceleration begins. Higher offset = slow movements feel completely unaccelerated. Useful for micro-adjustments.
Limit Maximum multiplier cap. The curve asymptotically approaches this value. 2.0 = fast flick is 2× your slow sensitivity.
Gain When ON, smooths the acceleration near the limit so there's no abrupt cap. Recommended for most users.
Y/X Ratio Scales vertical vs horizontal sensitivity. Use 0.85–0.95 to reduce vertical flicking if you have wrist deviation.
Rotation Rotates the input vector. Compensates for mousepad angle or forearm alignment. ±5° is usually enough.
eDPI Tier Guide (Valorant)

Recommended Workflow
1. Set your DPI and current in-game sens above.
2. Pick the play style that fits your role.
3. The tool auto-tunes decay / offset / limit for your sens tier.
4. Use the Curve tab to visualize exactly what will happen.
5. Copy the config and paste it into Raw Accel.
6. Test in aim trainers (KovaaK's / Aimlabs) before ranked.
Curve Type Comparison
Natural — Exponential approach to limit. Smoothest feeling, most popular. Best all-rounder.
Linear — Constant gain per speed unit. Very predictable. Good for CS2 veterans.
Classic — Old-school quake-style power curve. High ceiling, less control at low speeds.
Power — Exponential power law. Very aggressive at high speeds. Not recommended for beginners.
Motivity — Combines offset + natural. Best micro-precision + flick power balance.
